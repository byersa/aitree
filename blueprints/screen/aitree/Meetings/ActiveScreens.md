# ActiveScreens

## Description
A screen that lets the user select from "instance" AgendaContainers to edit by MainInstancePage
ActiveScreens operates as a split screen manager.
MainInstancePage screen are activated and deactivatd by the AgendaContainerSelectPage

This screen will begin the use of `transferDataMap` for data exchange.
